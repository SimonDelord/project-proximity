# BHP Project Proximity

A proximity detection and collision avoidance system for mining fleet operations.

## Overview

This project implements real-time proximity monitoring for mining vehicles to enhance safety and prevent collisions in mining operations.

## Project Structure

```
BHP-project-proximity/
├── configs/          # Configuration files (Kubernetes, connectors, etc.)
├── scripts/          # Utility and deployment scripts
├── data/             # Data files, coordinates, sample datasets
├── docs/             # Documentation and guides
├── src/              # Source code
├── tests/            # Test files and test utilities
└── monitoring/       # Monitoring dashboards and alerting configs
```

## Features

- Real-time vehicle proximity detection
- Collision avoidance alerts
- Fleet tracking integration
- Safety zone monitoring
